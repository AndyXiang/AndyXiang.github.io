前言

Hexo+Github创建博客

    花了两天时间创建了现在这个博客，现在写一篇Blog记录下过程和遇到的问题。Github提供了免费的域名 github.io，只需向仓库中放入html文件就可以生成一个网页。Hexo是一个快速、简洁且高效的博客框架，渲染markdown文件快速生成html文件，再使用git来将Hexo生成html文件上传至仓库。

系统介绍

    我使用的系统是macOS Monterey 12.3.1，Hexo 5.4.2，git 2.36.1，npm 8.5.5。

创建github网页

    先注册一个github网站账号，以Name为例。在仓库（Repositories）中创建第一个仓库：Name.github.io。仓库名的形式需要如此（我因为已经创建过这个名字的仓库所以会报错）。对于免费账户，仓库需要创建为public。README文件可以勾选也可以不勾选。

    在创建好仓库后，进入仓库页面。现在要先做两个准备工作。先新建一个分支命名为master，并设置为default。其次，进入Settings页面，拉到最下面后，在左边一栏找到Github Page选项。
